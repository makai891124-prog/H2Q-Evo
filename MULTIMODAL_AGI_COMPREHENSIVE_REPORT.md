### Hamilton 积跨模态融合

```python
def fuse(q1, q2):
    """Hamilton 积保持 S³ 流形结构."""
    w = w1*w2 - x1*x2 - y1*y2 - z1*z2
    x = w1*x2 + x1*w2 + y1*z2 - z1*y2
    y = w1*y2 - x1*z2 + y1*w2 + z1*x2
    z = w1*z2 + x1*y2 - y1*x2 + z1*w2
    return normalize([w, x, y, z])
```

### Berry 相位对齐度量

- 模态间相位一致性: `θ = arccos(|q1·q2|)`
- 用于跨模态匹配评分

### Fueter 正则性约束

- 确保四元数函数满足全纯性条件
